1. Create a JSP page to accept a number from a user and display it in words: Example:
123 – One Two Three. The output should be in red color



apache-tomcat
 └── webapps
      └── numberapp
           └── number.jsp



number.jsp


<html>
<body>

<h2>Number to Words</h2>

<form method="post">
Enter Number:
<input type="text" name="num">
<br><br>
<input type="submit" value="Convert">
</form>

<%
String n = request.getParameter("num");

if(n != null)
{
String words[]={"Zero","One","Two","Three","Four","Five","Six","Seven","Eight","Nine"};

out.print("<p style='color:red;'>");

for(int i=0;i<n.length();i++)
{
char c=n.charAt(i);
int digit=c-'0';
out.print(words[digit]+" ");
}

out.print("</p>");
}
%>

</body>
</html>



Open in browser
http://localhost:8080/numberapp/number.jsp

Example input

123

Output (red color)

One Two Three







2.Write a java program to blink image on the JFrame continuously



Save as BlinkImage.java

import javax.swing.*;
import java.awt.*;

public class BlinkImage extends JFrame implements Runnable {

    JLabel label;
    Thread t;

    BlinkImage() {

        ImageIcon img = new ImageIcon("image.jpg");

        label = new JLabel(img);

        add(label);

        setSize(400,300);
        setVisible(true);
        setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);

        t = new Thread(this);
        t.start();
    }

    public void run() {

        try {
            while(true) {

                label.setVisible(false);
                Thread.sleep(500);

                label.setVisible(true);
                Thread.sleep(500);
            }
        }
        catch(Exception e) {
            System.out.println(e);
        }
    }

    public static void main(String args[]) {
        new BlinkImage();
    }
}




Example folder:

BlinkImage.java
image.jpg

How to run program

Compile
javac BlinkImage.java

Run
java BlinkImage
