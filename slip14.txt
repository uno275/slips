Write a Java program for a simple search engine. Accept a string to be searched. Search
the string in all text files in the current folder. Use a separate thread for each file. The
result should display the filename and line number where the string is found. 



Create some text files

Example:

file1.txt

hello java
welcome to programming

file2.txt

java is powerful
thread programming example



import java.io.*;
import java.util.*;

class SearchThread extends Thread {
    File file;
    String search;

    SearchThread(File file, String search) {
        this.file = file;
        this.search = search;
    }

    public void run() {
        try {
            BufferedReader br = new BufferedReader(new FileReader(file));
            String line;
            int lineNo = 0;

            while ((line = br.readLine()) != null) {
                lineNo++;
                if (line.contains(search)) {
                    System.out.println("Found in file: " + file.getName()
                            + " at line: " + lineNo);
                }
            }

            br.close();
        } catch (Exception e) {
            System.out.println(e);
        }
    }
}

public class SearchEngine {
    public static void main(String[] args) throws Exception {

        Scanner sc = new Scanner(System.in);
        System.out.print("Enter string to search: ");
        String search = sc.nextLine();

        File folder = new File(".");

        File files[] = folder.listFiles();

        for (File f : files) {
            if (f.getName().endsWith(".txt")) {
                SearchThread t = new SearchThread(f, search);
                t.start();
            }
        }
    }
}




javac SearchEngine.java
java SearchEngine

Example input

Enter string to search: java

Output

Found in file: file1.txt at line: 1
Found in file: file2.txt at line: 1
