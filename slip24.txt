1. Write a java program to scroll the text from left to right continuously. 



import java.awt.*;
import java.awt.event.*;

public class ScrollText extends Frame implements Runnable {

    TextField tf;
    Thread t;
    String msg = "Java Programming ";

    ScrollText() {

        tf = new TextField(30);
        tf.setText(msg);

        add(tf);

        setSize(400,150);
        setVisible(true);

        t = new Thread(this);
        t.start();

        addWindowListener(new WindowAdapter() {
            public void windowClosing(WindowEvent we) {
                dispose();
            }
        });
    }

    public void run() {

        try {

            while(true) {

                msg = msg.substring(1) + msg.charAt(0);

                tf.setText(msg);

                Thread.sleep(200);

            }

        }
        catch(Exception e) {
            System.out.println(e);
        }
    }

    public static void main(String args[]) {

        new ScrollText();

    }
}




Compile
javac ScrollText.java
Run
java ScrollText
Output

A window will open and text will scroll continuously.

2. Write a JSP script to accept username and password from user, if they are same then
display “Login Successfully” message in Login.html file, otherwise display “Login
Failed” Message in Error.html file. 


apache-tomcat
  webapps
      loginapp
          index.html
          login.jsp
          login.html
          error.html

<html>
<body>

Username:

<input type="password" name="pass"><br><br>

<input type="submit" value="Login">

</form>

</body>
</html>
