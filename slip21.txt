Write a java program to accept ‘N’ Subject Names from a user store them into
LinkedList Collection and Display them by using Iterator interface. 

Save as SubjectList.java

import java.util.*;

public class SubjectList {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        LinkedList<String> subjects = new LinkedList<String>();

        System.out.print("Enter number of subjects: ");
        int n = sc.nextInt();
        sc.nextLine();

        System.out.println("Enter subject names:");
        for (int i = 0; i < n; i++) {
            String sub = sc.nextLine();
            subjects.add(sub);
        }

        System.out.println("Subject names are:");
        Iterator<String> it = subjects.iterator();

        while (it.hasNext()) {
            System.out.println(it.next());
        }

        sc.close();
    }
}



Compile:

javac SubjectList.java

Run:
java SubjectList

Example input
Enter number of subjects: 4

Enter subject names:
Java
Python
DBMS
OS

Output

Subject names are:
Java
Python
DBMS
OS






2. Write a java program to solve producer consumer problem in which a producer
produces a value and consumer consume the value before producer generate the next
value. (Hint: use thread synchronization) 


Save as ProducerConsumerDemo.java



class Q {
    int n;
    boolean valueSet = false;

    synchronized void put(int n) {
        try {
            while (valueSet)
                wait();

            this.n = n;
            System.out.println("Produced: " + n);
            valueSet = true;
            notify();
        } catch (Exception e) {
            System.out.println(e);
        }
    }

    synchronized void get() {
        try {
            while (!valueSet)
                wait();

            System.out.println("Consumed: " + n);
            valueSet = false;
            notify();
        } catch (Exception e) {
            System.out.println(e);
        }
    }
}

class Producer extends Thread {
    Q q;

    Producer(Q q) {
        this.q = q;
    }

    public void run() {
        for (int i = 1; i <= 5; i++) {
            q.put(i);
        }
    }
}

class Consumer extends Thread {
    Q q;

    Consumer(Q q) {
        this.q = q;
    }

    public void run() {
        for (int i = 1; i <= 5; i++) {
            q.get();
        }
    }
}

public class ProducerConsumerDemo {
    public static void main(String[] args) {
        Q q = new Q();

        Producer p = new Producer(q);
        Consumer c = new Consumer(q);

        p.start();
        c.start();
    }
}


Compile:
javac ProducerConsumerDemo.java

Run:
java ProducerConsumerDemo

Consumed: 1
